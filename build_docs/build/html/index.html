PELE Platform¶

Pele platform is a suit of packages that enables users to find novel solutions to drug discovery problems in matter of hours.

PELE packages:

  • Docking Refinement (1-2h)

  • Pose Scorer (1-2h)

  • Pocket Exploration (24h)

  • Protein-Protein Inhibitors (10h)

  • HT-Fragment Growing (30min/fragment)

  • GPCR orthosteric site exploration (5h)

  • Binding Events (6h)

  • Biased Simulation (10h)

  • Conformational change simulation (6h)

  • Minimization (30min)

Github : https://github.com/NostrumBioDiscovery/pele_platform
